# Current Implementation Plan

## Getting Started

This file should contain your implementation plan.

### To set up:

1. Copy `template-plan.md` to `current-plan.md` (replacing this file)
2. Or create your own plan following the template structure
3. Customize phases, tasks, and acceptance criteria

### Plan Structure Required:

```markdown
# Project Name Implementation Plan

## Plan Metadata
- Version: x.x
- Status: in_progress
- Current Phase: N

## Constitution
[Rules that never break]

## Phase 1: [Name]
**Status**: pending | in_progress | completed
**Gate**: automatic | human_approval
**Checkpoint**: [What must be true]

### Tasks
- [ ] Task with acceptance criteria

## Phase 2: ...
```

### Once ready:

Run `/start-phase 1` to begin!
